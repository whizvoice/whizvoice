# Whiz Voice

Whiz Voice is built to be the hands-free computing interface of the future.
It is integrated with Claude and Asana, and allows users to open and interact with other apps on the device.
Users can ask questions, add things to their Asana task list, or perform operations on their phone like sending messages or using apps, through voice chat or texting on Whiz Voice.
The Android app can be opened with wake word "Hey Whiz", or, if it is set as the default assistant, with long press power button.
The app is integrated with Google Maps, YouTube Music, WhatsApp, and Google Messages (for SMS texting). It also has some common assistant functions like setting alarms and getting the weather.
