# Whiz Voice

Whiz Voice is built to be the hands-free computing interface of the future.
It uses Claude for it's LLM to be able to answer user questions or help make decisions. Available tools include random color picker, weather, and saving preferences such as contact nicknames.
It's integrated with the Asana API for to do list management.
It also has screen agent functions for Youtube Music, Google Maps, WhatsApp, and Google Messages, meaning that it can open and interact with these apps on the device.
It also has some common assistant functions like managing alarms or timers, creating calendar invites, and getting the weather.
The Android app can be opened with wake word "Hey Whiz", or, if it is set as the default assistant, with long press power button.
